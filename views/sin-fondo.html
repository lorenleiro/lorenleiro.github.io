Mons

A mobile game where you need to score higher each time.

This was my first attempt on making something simple but at the same time fun. It was my firsy time developing an app for mobile, so I encountered more problems that I needed to fix. All the 3d models where made by myself, so it was quite satisfactory when I added all together an see that everything clicks.

Main features

  • Procedural generated platforms and backgrounds.

  • Increase difficulty over distance.

  • Customizable character.

  • Smooth and relaxing atmosphere.

What was the challenge?

The main challenge here was a way to keep the player focused on the jumps, so each jump should be interesting, but not over complicated. I came with a way that keeps the player progress and makes the game more difficult when he reaches a specific height. This was awesome, eveytime the player had e new challengue.
