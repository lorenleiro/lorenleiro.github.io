Cowboy Clicker

As an intergalatic cowboy bountry hunter, click and purchase upgrades to become the most successful (and the richest) bounty hunter there is!

This game was created for the 2023 Clicker Game Jam in 7 days.

To test my skills in a short period of time, I took part in the Clicker Jam 2023, where we worked with a group of people from around the world to create a super fun clicker with lots of story and content for the player to enjoy as they progressed through the game. The theme of the jam was Interdimensional, so once we knew the theme, we let our imaginations run wild. I was tasked with programming all the game mechanics. In addition to developing the entire loop, I was also responsible for creating the user interface using the new Unity UI Toolkit, which allowed me to create an interactive interface with very little effort.

Main features

  • History progress unlock

  • Passive attack and income

  • Lost of unique upgrades

  • Two modes: horde fight and boss fight

  • Lost of levels to navigate back and forth

  • Developed with ScriptableObjects to make new asset creating faster and easier

  • UI made with the new Unity UI Toolkit

  • Game state save and load

What was the challenge?

To make the game entertaining, we focused not only on the combat system, but also added a story that seemed disjointed at first, but made more sense as the player progressed, until the final level where everything made sense. To create all this content, I focused on implementing a system that would allow us to create new content efficiently and easily. I decided to use Unity Scriptable Objects for this task. This made it very easy to create and edit content and allowed us to create more levels than we had planned. As well as using it for content, we also used it to edit the values of combat, enemies and bosses to make the game easier to balance.

Project duration: 09/01/2023 - 16/01/2023
