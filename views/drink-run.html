Drink Run

Try to find all of your gun missing pieces so you can complete reach the final destiny...

This game was created for the Metroidvania Month 18 Jam in 31 days.

This time all was different for me, I tried to find a international people and try to work as a team. I found a team with really cool people and we started creating what is now called Drink Run. The theme of the Jam was Night Life, so we wanted to add that to our game. We came across many difficulties but in the end we made a finish product that we can show to other people.

Main features

  • Multiple gun upgrades

  • Interactable stations where you can buy drinks or save your game

  • Three different areas with different aesthetics

  • Varios types of enemies with different behaviors

  • Final level

  • UI made with the new Unity UI Toolkit

What was the challenge?

One of the main challenges here was to made the player fun to play. I create a movement script that makes the player responsive with a lot of features like dashing, double jump or coyote jump. The weapon system in this game made for us the posibility of creating various different gun settings without any single additional line of code. Another big obtacle was the save and load system, that was my first time working with unique GUIDs and generating each level with the data the player has collected.

Project duration: 10/10/2022 - 10/16/2022
