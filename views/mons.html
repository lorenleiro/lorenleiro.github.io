Mons

A mobile game where you have to climb up platforms in a floating island world.

Mons is very easy to understand, and you only have to do one thing, go as high as you can. I originally developed it for PC, but then I realized that the game would be much more fun if I ported it to mobile. I created all the 3D models myself, and it was very satisfying when I put it all together and it was actually fun to play.

Main features

  • Procedural generated platforms and backgrounds

  • Multiple platforms with different behaviors and shapes

  • Adjustable jump grants the player more control

  • Increase difficulty over height

  • Customizable character

  • Smooth and relaxing atmosphere

  • 2.5D style game

What was the challenge?

The biggest challenge was to keep the player focused on the jumps so that each jump is interesting but not too complicated. I found a way to make the game more difficult and keep the player interested. This was done by checking the actual height of the player and setting the difficulty of the game to the right value (new platforms, more distance, less reaction time...). This was ingenious, because it offered the player a new challenge every time.

Project duration: 14/12/2021 - 09/01/2022
